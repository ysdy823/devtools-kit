9 Essential Developer Tools

Everything you need for day-to-day development — fast, private, and always available.

{ }

JSON Formatter

Pretty-print, minify, and validate JSON with syntax highlighting. Instantly catch malformed payloads and explore nested structures.

B64

Base64 Encoder / Decoder

Encode and decode Base64 strings in real time. Handles text and binary data for API payloads, authentication tokens, and data URIs.

JWT

JWT Decoder

Inspect JSON Web Tokens — decode header, payload, and signature. See expiration times, claims, and issuer details at a glance.

#

Hash Generator

Generate MD5, SHA-1, SHA-256, and SHA-512 hashes instantly. Verify checksums and compare file integrity without leaving your browser.

ID

UUID Generator

Generate RFC 4122 v4 UUIDs — single or in bulk. Copy-ready output for database seeding, testing, and unique identifier needs.

.*

Regex Tester

Build and test regular expressions with real-time match highlighting. See capture groups, test against sample text, and iterate fast.

%

URL Encoder / Decoder

Encode and decode URL components. Handle query parameters, special characters, and percent-encoded strings effortlessly.

T

Timestamp Converter

Convert between Unix timestamps and human-readable dates. Supports multiple timezones and common date formats.

Δ

Text Diff

Side-by-side text comparison with highlighted additions, deletions, and changes. Perfect for comparing configs, API responses, and code snippets.
